S.F. State faculty set December strike date, joining other CSU campuses
-----
The union representing California State University faculty announced Thursday that members from four campuses will go on one-day labor strikes in December if a new contract is not settled by then.The four campuses that will hold one-day strikes are Cal Poly Pomona (Dec. 4), San Francisco State (Dec. 5), Cal State Los Angeles (Dec. 6) and Sacramento State (Dec. 7).
