Porch pirates, begone: How to prevent package theft
-----
Three things to know about porch pirates* Porch pirates target package deliveries so they can sell the contents.
* Package tracking, delivery to a store or Amazon locker, and requiring a signature can proactively stop package theft.
* Home security cameras, video doorbells, and home security systems can deter theft as well as make it easier to identify the offender if someone still takes your package.
